
Platilla Latex para la elaboración de memorias o trabajos del Máśter TWCAM.

Se proporciona un ejemplo de uso de esta plantilla.

Contenido:

  - Plantilla: en el directorio cls está el fichero twcam-trabajo.cls
    (no es necesario editar este fichero)

  - Ejemplo de uso de la plantilla: ejemplo-trabajo.tex  (hay que editar
    este fichero para poner: autor/a, título, asignatura y el contenido).

  - Directorio para las figuras: figs (no se debe cambiar el nombre,
    se pueden almacenar en este directorio las figuras en formato
    pdf, png o jpg)

  - Directorio para la bibliografía: bib (no se debe cambiar el nombre,
    hay que añadir elementos al fichero bibliografia.bib)
